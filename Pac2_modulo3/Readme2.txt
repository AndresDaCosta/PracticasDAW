10.	Realiza un programa que permita al gerente de una cooperativa introducir el número de kilos de patatas 
	que vende a un cliente, y el precio al que se encuentra este producto. El programa calculará el precio 
	en bruto del pedido. 
•	Si este precio supera los 50€ entonces el descuento que aplicará será del 10%. 
•	Si el precio supera los 100€ entonces el descuento será del 20%. Después, el gerente indicará que tipo 
	de cliente es. Si introduce “PARTICULAR” entonces al precio con descuento debe aplicarse el 21% de IVA. 
	Mientras que si introduce “EMPRESA” el precio final coincidirá con el precio bruto después de aplicar 
	el descuento pertinente. 
El programa debe mostrar:  
•	El precio bruto 
•	El precio con descuento 
•	El precio final 
 
Nota: En este ejercicio se permite utilizar la función equals para las cadenas.
